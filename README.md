# Дипломный проект: Информационная платформа для публикации учебного видеоконтента студентов

# О проекте
Разработан веб-сайт, позволяющий студентам публиковать свои учебные видеоматериалы (лабораторные работы, проекты, демонстрацию навыков), формировать видео-портфолио, а преподавателям — просматривать, комментировать и оценивать работы.

# Функциональные возможности
- Регистрация и авторизация пользователей
- Разграничение ролей: студент, преподаватель, администратор
- Загрузка видео (форматы MP4, WebM, до 100 МБ)
- Просмотр ленты видео с поиском и фильтрацией (новые/популярные)
- Система комментариев и лайков (без перезагрузки страницы, AJAX)
- Личный кабинет (управление своими видео)
- Админ-панель (управление пользователями и контентом)

# Технологии
- **Backend:** PHP
- **База данных:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript
- **Интерактивность:** AJAX (Fetch API)
- **Сервер:** MAMP (Apache, MySQL)

# Установка и запуск

1. Установите MAMP (или OpenServer/XAMPP)
2. Скопируйте папку `video_portfolio` в `htdocs`
3. Импортируйте базу данных из файла `database.sql` (см. раздел 2.1 диплома)
4. Настройте подключение к БД в `config/database.php` (порт 8889 для MySQL в MAMP)
5. Откройте в браузере: `http://localhost:8888/video_portfolio/`
